Project
ARNI Project

Category: QlikSense

ARNI is one of the supply chain applications. Currently I'm working on this application. It is a development application from scratch. I flowed the 3-tier architecture, such as extract, transform, and final application. I'm gathering the data from the SAP, Snowflake, and flat files in the extract application, then storing it into the qvd for faster retrieval in the transform application. In the transform application, I'm loading all those QVD converted data and applying the transformation as per the client requirement. After applying the transformation, then storing the data in QVD format for the final load. In the final application, I'm loading transformed QVD and creating the data model based on the table key. Once final application scripting is complete, I'll create the charts as per the client's request.
